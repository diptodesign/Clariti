📘 Documentation

Everything you need to install, configure, and master AccessVeil Pro.

🚀 Installation

  1. Download the plugin – Get accessveil-pro.zip from your Envato purchase downloads.
  2. Upload to WordPress – Go to Plugins → Add New → Upload Plugin. Choose the zip file and click Install Now.
  3. Activate – Click Activate Plugin. You'll see "AccessVeil Pro" in your admin sidebar.
Tip: After activation, the plugin creates default settings automatically. You can start using the widget immediately after license activation.

🔑 License Activation

AccessVeil Pro requires a valid license to unlock all 13 modules. Navigate to AccessVeil Pro → License in your WordPress admin.

  • Enter your Envato purchase code (format: AVPRO-XXXXX-XXXXX-XXXXX-XXXXX)
  • Click Activate License

⚡ Quick Start

Once license is active:

  • Visit AccessVeil Pro → Modules – toggle which accessibility tools you want to offer.
  • Ensure "Enable Widget" is checked.
  • Visit your site frontend – you'll see the accessibility button.

⚙️ Global Settings

Located under the Modules tab (scroll to bottom):

SettingDescription
Enable WidgetMaster on/off switch for the frontend widget.
Hide on MobileAdds CSS to hide the widget on screens smaller than 768px.

🎨 Appearance

Customize the widget look under AccessVeil Pro → Appearance.

  • Position: bottom-right (default), bottom-left, top-right, top-left.
  • Brand Color: Hex color picker – affects button, toggles, highlights.
  • Widget Size: small, medium, large.
  • Custom CSS: Add your own overrides (see below).

✏️ Custom CSS

Use the textarea in Appearance to inject custom styles. All widget elements are prefixed with .avp- or #avp-widget.

/* Example: make trigger larger */
#avp-widget .avp-trigger {
    width: 80px !important;
    height: 80px !important;
}

⚡ Hooks & Filters (Developers)

Extend AccessVeil Pro programmatically:

// Filter settings before render
add_filter('avp_settings', function($settings) {
    $settings['widget_color'] = '#dc2626';
    return $settings;
});

// Filter module list
add_filter('avp_modules', function($modules) {
    // modify modules array
    return $modules;
});

📎 Shortcode

You can manually place the widget trigger using shortcode [avp_trigger]. Useful for custom header placements.

🌐 Translation

AccessVeil Pro is translation-ready. Use .po/.mo files in /languages folder. Text domain: accessveil-pro.

🛠️ Troubleshooting

  • Widget not appearing? Verify license is active and "Enable Widget" is checked.
  • Modules not working? Clear any caching plugins. Check browser console for JS errors.
  • Conflict with theme? Use Custom CSS to adjust z-index or positioning.
  • Screen reader not speaking? Ensure site uses HTTPS (Web Speech API requirement).

❓ FAQ

Q: Can I use this on multiple sites?
A: One regular license covers one WordPress installation (Envato terms).

Q: Does it slow down my site?
A: No. Assets are minified and only loaded when widget is enabled. Zero impact on Core Web Vitals.

Q: Is it GDPR compliant?
A: Yes, no personal data is stored. Preferences are saved in sessionStorage.
