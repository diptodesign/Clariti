Documentation

Everything you need to install, configure, and master Clariti.

Installation

  1. Download clariti.zip from your Envato purchases
  2. Go to Plugins > Add New > Upload Plugin
  3. Choose the zip file and click Install Now
  4. Click Activate Plugin

License Activation

Navigate to Clariti > License and enter your Envato purchase code. The plugin verifies your code and unlocks all 18 modules.

Without activation, the accessibility widget will not appear on your site.

Quick Start

  1. Go to Clariti > Modules
  2. Toggle which modules you want to offer visitors
  3. Ensure "Enable Widget" is checked
  4. Visit your site — the eye-icon button appears automatically
