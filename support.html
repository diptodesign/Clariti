Quick Answers

Frequently Asked Questions

How do I install AccessVeil Pro?

Download the plugin zip from your Envato purchases. In WordPress admin, go to Plugins → Add New → Upload Plugin, choose the zip file, and click Install. Then activate the plugin and enter your license key under AccessVeil Pro → License.

The widget is not showing on my site. What should I do?

Check three things: (1) Your license is active under AccessVeil Pro → License, (2) "Enable Widget" is checked under Modules → Global Settings, (3) You haven't enabled "Hide on Mobile" while testing on a phone. Also clear any caching plugins.

Can I use AccessVeil Pro on multiple sites?

A Regular License covers one WordPress installation. For multiple sites, you need to purchase additional licenses. An Extended License is available for SaaS or commercial applications. Check Envato's license terms for full details.

Does AccessVeil Pro slow down my website?

No. The plugin loads minimal assets (under 50KB total) and only when the widget is enabled. It has zero impact on Core Web Vitals. Scripts are deferred, CSS is optimized, and there are no external API calls.

Is AccessVeil Pro GDPR compliant?

Yes! AccessVeil Pro does not use cookies, does not track users, and does not send any data to external servers. User preferences are stored in sessionStorage (browser memory) and are automatically cleared when the browser is closed.

Will this work with my theme / page builder?

AccessVeil Pro works with any WordPress theme and all major page builders including Gutenberg, Elementor, Divi, WPBakery, Beaver Builder, and Oxygen. The widget injects at the footer level and uses standard CSS that doesn't conflict.

How do I update the plugin?

When a new version is released, download it from your Envato purchases. Then go to Plugins → Add New → Upload Plugin, upload the new zip, and WordPress will handle the update automatically. Your settings are preserved.

What if I need a refund?

Refunds are handled through Envato Market's refund policy. If the plugin doesn't work as described and we can't resolve the issue, you're covered by Envato's buyer protection. Contact us first — we'll do our best to fix any problem.
